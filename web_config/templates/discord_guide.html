Creating a Discord Bot

This guide will walk you through the steps to create a Discord bot and add it to your server.

Step 1: Create a Discord Application

  1. Go to the Discord Developer Portal and log in with your Discord account.
  2. Click the "New Application" button in the top right corner.
  3. Enter a name for your application (e.g., "Larry's Gym Tracker") and click "Create".
  4. Save your Application ID - you'll need this later.
Note: The Application ID is different from the Bot Token. The Application ID is used for inviting the bot to your server.

Step 2: Create a Bot User

  1. In the left sidebar, click on "Bot".
  2. Click the "Add Bot" button on the right.
  3. Confirm by clicking "Yes, do it!"
  4. Customize your bot's username and avatar (optional).
  5. Under the "TOKEN" section, click "Reset Token" and then "Yes, do it!".
  6. Copy the token that appears - this is your Bot Token.
IMPORTANT: Keep your bot token secret! It's like a password for your bot. Never share it publicly or commit it to version control. If your token is compromised, reset it immediately.

Step 3: Configure Bot Permissions

  1. In the Bot settings, scroll down to "Privileged Gateway Intents".
  2. Enable the following intents:
    • SERVER MEMBERS INTENT
    • MESSAGE CONTENT INTENT
  3. Save your changes.
Note: These intents are required for the bot to read message content and track server members.

Step 4: Create an Invite URL

  1. In the left sidebar, click on "OAuth2" and then "URL Generator".
  2. Under "SCOPES", select "bot" and "applications.commands".
  3. Under "BOT PERMISSIONS", select the following permissions:
    • Read Messages/View Channels
    • Send Messages
    • Send Messages in Threads
    • Embed Links
    • Attach Files
    • Add Reactions
    • Use External Emojis
    • Mention Everyone
    • Connect
    • Speak
  4. Copy the generated URL at the bottom of the page.

Step 5: Invite the Bot to Your Server

  1. Open the URL you copied in the previous step in your web browser.
  2. Select the server you want to add the bot to.
  3. Click "Authorize".
  4. Complete the captcha if prompted.
Success! Your bot should now appear in your server's member list.

Step 6: Get Your Server and Channel IDs

  1. Open Discord's User Settings (gear icon in the bottom left).
  2. Go to "Advanced" and enable "Developer Mode".
  3. To get your Server (Guild) ID:
    • Right-click on your server name
    • Select "Copy ID"
  4. To get Channel IDs:
    • Right-click on the text or voice channel
    • Select "Copy ID"
Tip: You'll need these IDs to configure your bot in the config.json file.

Step 7: Configure Your Bot

  1. Return to the configuration page and enter your:
    • Bot Token
    • Guild (Server) ID
    • Text Channel ID
    • Voice Channel ID
  2. Configure other settings as needed.
  3. Click "Save Configuration".
